# 🧠 SkillMap – Your Personalized Learning Tracker

SkillMap is a full-stack productivity tool to help you **track learning progress**, visualize **skill development**, and generate **AI-curated learning paths** based on your goals.

---

## ✨ Features

- ✅ Interactive learning dashboard
- ✅ Track subtopic completion with estimated hours
- ✅ Visual progress bar and analytics
- ✅ AI-powered topic generator based on learning goals
- ✅ User-friendly interface built with React and Tailwind CSS
- ✅ Backend API with Express.js and MongoDB
